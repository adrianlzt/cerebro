https://keybase.io

Especie de Dropbox pero encriptado.
Los ficheros se encriptan/desencriptan en local.


# Cli
sudo pacman -S keybase

# Uso
Una vez registrado podemos asociar a nuestro usuario una cuenta de twitter, github, reddit, coinbase, hackernews, http, https y dns

Buscar otros usuarios:
keybase search <nombre>
keybase id <id_user>
keybase id <id_user>@<servicio>
keybase track <id>
