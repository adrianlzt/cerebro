# Telescope
https://github.com/nvim-telescope/telescope.nvim#pickers

Nos permite hacer fuzzy finding para ficheros, grep, keymaps, etc


# Vim
Buscar palabra
/palabra

Buscar hacia atrás
&palabra

*/n siguiente resultado
#/p resultado anterior

Resaltar resultados:
:set hlsearch

Para desactivar el resaltado:
:set hlsearch!


Buscar un caracter en la misma línea:
fX
; siguiente match en la misma línea
