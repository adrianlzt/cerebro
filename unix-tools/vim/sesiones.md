Con este plugin, al arrancar vi sin nada, veremos los ficheros más recientes y debajo las sesiones guardadas.

Una sesión es una configuración de tabs/windows/buffers.

Comandos para usar las sesiones:
:SLoad    load a session
:SSave    save a session
:SDelete  delete a session
:SClose   close current session
