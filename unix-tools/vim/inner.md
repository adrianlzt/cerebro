Borrar el contenido de dentro de un delmiter.
La explicación es `Change all Inner "`
```
ci"
```

```
key: "foo baz bar"
```

Si hacemos `ci"` dentro de la cadena de texto, borraremos todo el contenido dentro de las comillas.

Similar, pero sin saltar a modo a modo insert:
```
di"
```

Este sería `Delete Inner "`
