http://snapper.io/

Es algo como el time machien de OSX.
Hace snapshots cada x tiempo, y nos permite ver las diferencias.

Para ver la configuración una opción sencilla es:
```bash
btrfs-assistant-launcher
```

Mostrar config:
```bash
snapper list-configs
```

Mostrar snapshots de todas las configs
```bash
snapper ls -a
```
