El schema es el formato de los predicates (datos).

Pueden ser:
  string
  float
  int
  geo
  dateTime
  bool
  uid

El tipo "uid" se usa para crear las relaciones (edges) con otros nodos.
Puede ser del tipo "[uid]" para indicar relaciones 1-N
