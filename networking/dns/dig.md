Obtener el AXFR (zone transfer) de un dominio
dig @ns.domain.com dominio.com AXFR


MX de un server
dig server.com MX


Servidor autoridad de un dominio
dig pepe.com SOA


Tipos registros:
A, AAAA, MX, AXFR, NS, SOA, TXT, PTR, CNAME
