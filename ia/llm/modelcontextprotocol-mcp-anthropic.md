# Clientes

## mcpgod

<https://github.com/mcpgod/cli>

Ver las tools que ofrece un MCP server:

```bash
npx -y mcpgod tools @modelcontextprotocol/server-everything
```

Usar una tool:

```bash
npx -y mcpgod tool @modelcontextprotocol/server-everything add a=59 b=40
```
