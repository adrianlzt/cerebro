Minimalistic coding agent written in Rust, optimized for memory footprint and performance

# Install

```bash
cargo install zerostack
```

Para tener sandboxing de comandos bash usar:
```bash
paru -S bubblewrap
```

# Config

```
$HOME/.config/zerostack/config.json
```
