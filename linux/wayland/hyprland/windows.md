# Cambiar layout

Cambiar disposición vertical/horizontal

SUPER + J

# Grupos

Crear o borrar grupo. Las ventanas que creemos se meterán como tabs

SUPER + G

Para meter una ventana en un grupo, "empujarla" con:

SUPER + CONTROL + cursores

# Float

SUPER + v

# Move

SUPER + click

# Redimensionar
SUPER + click derecho

SUPER + SHIFT + cursores

# Cerrar

SUPER + SHIFT + q
