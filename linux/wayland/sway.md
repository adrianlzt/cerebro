Reload:
```bash
sway reload
```

O también con: mod+shift+c

Ejecutar comandos de sway:
```bash
swaymsg '[app_id="notas_term"] scratchpad show'
```

Obtener el tree de ventanas:
```bash
swaymsg -t get_tree
```

# escala / scale

Si las cosas se ven muy grandes, es porque se tiene configurada un "scale" a la pantalla.

Esto se hace porque las pantallas tienen mucha definición para el tamaño que tienen.
