http://www.docker.io/gettingstarted
Terminal emulada donde aprender a usar docker.

Docker tiene un cliente y un servidor.

Los comandos son: docker <cmd>

Ver versión, del cliente y el servidor (por defecto conectará a 127.0.0.1:4243):
docker version


Como sería con los módulos de puppet, aquí podemos buscar container ya fabricados:
docker search <string>


docker pull nombre

docker run nombre
