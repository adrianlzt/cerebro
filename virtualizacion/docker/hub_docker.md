Repo de imagenes de docker: hub.docker.com

Nos permite crear una imagen privada de forma gratuita.
Si necesitamos más podemos usar varios tags con el mismo repo y/o branches.


# Hooks
Podemos confiurar un webhook que será disparado cuando se haga un build de una imagen.

Ejemplo de un hook:
https://docs.docker.com/docker-hub/webhooks/
