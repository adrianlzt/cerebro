https://www.docker.io

Docker sirve de capa intermedia entre el OS y una aplicación empaquetada y autosuficiente.
Es un escalón intermedio entre paquetes RPM y máquinas virtuales.

Creas tu aplicación con todo lo necesario, la empaquetas con docker, y la distribuyes.
Cualquier entorno (dev, cloud, produccion, VM, etc) que tenga instalado Docker puede ejecutar esta aplicación sin necesidad de preocuparse por dependencias, etc.
