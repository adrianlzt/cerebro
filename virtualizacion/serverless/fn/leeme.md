http://fnproject.io/

The Fn project is a container native serverless platform that you can run anywhere -- any cloud or on-premise. It’s easy to use, supports every programming language, and is extensible and performant.

Una cli que detecta el language de tu proyecto (función), lo empaqueta en un contenedor docker y lo despliega asociandole una url relativa a tu app general (URL/r/app/funcion)

El esquema es parecido a lo que hace S2I (source2image) en openshift. Aquí simplificado para solo adaptarse a este caso.
