http://pusher.com/

Conectividad bidireccional realtime entre el server y los clientes usando websockets.

Un caso de uso son listas publish/subscribe.

Otro caso serían un chat con "presencia". El servidor va enviando a los clientes cuando hay alguien conectado o desconectado.

También podemos enviar notificaciones en real time a los usuarios.

Hacer un realtime chat fácilmente.

Más ejemplos: https://pusher.com/tutorials
