https://github.com/nginx/unit

universal web app server – a lightweight and versatile open source server project that works as a reverse proxy, serves static assets, and runs applications in multiple languages.

Puede ejecutar aplicaciones de python, php, go, webassembly, etc, sin ninguna otra app (quitamos php-fpm, gunicorn, etc).

Permite (o más bien, solo permite) configuración via API-JSON.
